# Property Calc

Free Australian property calculators and mortgage tools.

**Tagline:** Every number you need before you buy  
**Domain:** realestatecalculators.com.au

## Tools included

- Stamp Duty Calculator
- Borrowing Calculator
- Rental Yield Calculator
- Cash Flow Calculator
- LMI Estimator
- Offset Account Calculator
- CGT Calculator
- Depreciation Calculator
- Land Tax Calculator
- Deal Scorer

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for production

```bash
npm run build
```

The static site is exported to the `/out` directory, ready to deploy to any static host.
